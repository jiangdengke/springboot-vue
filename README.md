## 项目介绍
本项目是由`SpringBoot3+Vue3+MybatisPlus`等技术做的家具后台管理系统
## 用到的技术栈
### 后端技术栈
&emsp;1.SpringBoot<br/>
&emsp;2.Mybatis-Plus<br/>
&emsp;3.MySQL<br/>
### 前端技术栈
&emsp;1.Vue3<br/>
&emsp;2.ElementPlus<br/>
&emsp;3.Axios<br/>
## 快速部署
&emsp;1.clone到本地<br/>
&emsp;2.安装前端所需环境（node，axios,cnpm等）<br/>
&emsp;3.项目比较简单，无需其他配置，直接运行即可<br/>
